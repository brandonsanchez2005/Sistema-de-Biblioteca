Módulo de Devoluciones

Este módulo permite registrar devoluciones de libros, validar préstamos activos y gestionar multas por retraso.

Validaciones del Sistema

Préstamo Existente

El sistema verifica que exista un préstamo activo antes de permitir la devolución.

Prevención de Duplicados

No se permite registrar una devolución dos veces.

Control de Multas

Si pasan más de 30 días sin devolución, el sistema genera una multa automáticamente.
